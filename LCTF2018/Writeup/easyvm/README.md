603080开始是三段bytecode

sub_4009D2函数分三次对三段bytecode操作，sub_401722和sub_4017C2是对寄存器的赋值与还原，中间的sub_401502函数是操作函数，详细分析bytecode，可以得出它的操作过程：

1.计算输入长度，校验是否等于0x1C

2.将输入的每一位ch进行如下操作:

```
ch=((ch*0x3f)+0x78)%0x80
```
