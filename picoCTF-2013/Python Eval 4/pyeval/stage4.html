Task 4

Now that you've learned the basics, it's time to let you loose on something that's not as much of a toy example.

We are running a Python web server that uses eval to extract values from a query string. It's possible to exploit this server by sending it a maliciously crafted URL request. However, since you aren't given direct access to an eval call, you may have to fool around a bit before you can get eval to run on the right string.

The web server is running at python.picoctf.com:6364; you can navigate to http://python.picoctf.com:6364/index.html to see it in action and download its source. You'll probably want to know a bit about the HTTP protocol and connect with nc python.picoctf.com 6364 to actually exploit it. As with task 3, your goal is to get a shell and read out a file with the flag in it.
